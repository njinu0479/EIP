## 원형 큐의 인덱스 계산
- rear = (rear + 1) % SIZE: 데이터를 넣을 때 인덱스가 배열 크기를 넘어가면 다시 0으로 돌아오게 한다.
- front = (front + 1) % SIZE: 데이터를 뺄 때도 마찬가지로 인덱스가 순환한다.


## FIFO
- 가장 먼저 들어간 데이터가 가장 먼저 나오는 구조이다. 
- 하지만 중간에 deq()을 한 번 실행했기 때문에, 처음에 넣었던 1은 이미 사라진 상태이다.
- 따라서 그 다음에 들어갔던 2와 3이 차례대로 first와 second 변수에 담기게 된다.


## 최종 연산 및 출력
```c
printf("%d 그리고 %d", 2, 3);
```
2 그리고 3


## 최종출력결과
2 그리고 3<br/>
<br/>
⁕ printf 문구에 그리고라는 글자가 포함되어 있다. 답안을 작성할 때 숫자만 적지 말고 전체 출력 형식인 2 그리고 3을 정확히 적어야 감점을 피할 수 있다.<br/>
⁕ 큐가 가득 찼는지(Full) 비었는지(Empty) 확인하는 로직은 생략되어 있지만, 인덱스 변화 과정만으로도 충분히 풀 수 있는 문제이다.
