## 초기 설정 및 딕셔너리 생성
```py
lst = [1, 2, 3]
dst = {i : i * 2 for i in lst}
```
dst는 {1: 2, 2: 4, 3: 6}이 된다.


## 세트(Set) 생성
```py
s = set(dst.values())
```
- dst.values()는 [2, 4, 6]이고, 이를 세트로 만들었으니 s는 {2, 4, 6}이 된다.


## 리스트 수정정
```py
lst[0] = 99
```
- lst를 바꾼다고 해서 이미 만들어진 dst가 바뀌지는 않는다.


## 딕셔너리 값 수정 및 세트 요소 추가
```py
dst[2] = 7
s.add(99)
```


## 교집합 계산 및 길이 출력
```py
print(len(s & set(dst.values())))
```
- s = {2, 4, 6, 99}
- set(dst.values()) = {2, 6, 7}
- 두 집합에 공통으로 들어있는 요소는 {2, 6} 두 개
- 요소가 2개이므로 결과는 2가 출력


## 최종출력결과
2<br/>
<br/>
⁕ dst를 만들 때 lst를 사용했지만, 결과값은 독립적인 정수(int)로 저장됨.<br/>
⁕ &: 교집합, |: 합집합, -: 차집합<br/>
⁕ dst[2]=7로 인해 기존에 있던 4가 사라지고 7이 들어왔다는 점이 교집합 결과에 결정적인 영향을 줌.
