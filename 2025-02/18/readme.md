## func("BEST") 호출
- 이 함수는 문자열을 하나씩 읽어서 새로운 노드를 만드는데, 핵심은 n->p = h; 와와h = n;이다. 
- 새로운 노드를 항상 리스트의 맨 앞(Head)에 붙이는 방식(Head Insertion)이다.
1. h는 NULL이었으므로, [B] -> NULL이 되고 h는 B를 가리킨다.
2. 새로운 노드 E의 다음(p)을 현재 h(B)로 설정한다. 결과: [E] -> [B] -> NULL. h는 이제 E를 가리킨다.
3. S의 다음을 E로 설정한다. 결과: [S] -> [E] -> [B] -> NULL. h는 S를 가리킨다.
4. T의 다음을 S로 설정한다. 결과: [T] -> [S] -> [E] -> [B] -> NULL. 최종적으로 h는 T를 가리키며 반환된다.

## main 메서드의 출력 루프
- func에서 반환된 h(T노드 주소)를 가지고 while(n) 루프를 돈다.
1. n이 T를 가리킴 -> T 출력 -> n = n->p로 S로 이동 -> T 노드 메모리 해제(free)
2. n이 S를 가리킴 -> S 출력 -> n = n->p로 E로 이동 -> S 노드 메모리 해제
3. n이 E를 가리킴 -> E 출력 -> n = n->p로 B로 이동 -> E 노드 메모리 해제
4. n이 B를 가리킴 -> B 출력 -> n = n->p로 NULL로 이동 -> B 노드 메모리 해제
5. n이 NULL이므로 루프 종료.
