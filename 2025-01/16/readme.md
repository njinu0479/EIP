## func(data, 0, 4) (최상위 호출)
- st=0, end=4 -> mid = (0+4)/2 = 2
- 계산식: a[2] + Math.max(func(a, 0, 2), func(a, 3, 4))
- a[2] 값은 8이다.


## 왼쪽 자식 func(a, 0, 2) 계산
- st=0, end=2 -> mid = (0+2)/2 = 1
- 계산식: a[1] + Math.max(func(a, 0, 1), func(a, 2, 2))
- a[1] 값은 5이다.
    - func(a, 0, 1): mid=0 -> a[0] + Math.max(f(0,0), f(1,1)) = 3 + 0 = 3
    - func(a, 2, 2): st >= end 이므로 0 반환.
- 결과: 5 + max(3, 0) = 8


## 오른쪽 자식 func(a, 3, 4) 계산
- st=3, end=4 -> mid = (3+4)/2 = 3
- a[3] 값은 12이다.
    - func(a, 3, 3): 0 반환.
    - func(a, 4, 4): 0 반환.
- 결과: 12 + max(0, 0) = 12


## 최종 합산
- 다시 1단계로 돌아와서: a[2] + Math.max(왼쪽결과, 오른쪽결과)
- 8 + max(8, 12) = 8 + 12 = 20


## 요약
| 호출 범위 (st, end) |	mid | a[mid] | 좌/우 결과 중 max | 반환값 (a[mid] + max) |
| :---: | :---: | :---: | :---: | :---: |
| func(0, 1) | 0 | 3 | 0 | 3 |
| func(0, 2) | 1 | 5 | 3 | 8 |
| func(3, 4) | 3 | 12 | 0 |	12 |
| func(0, 4) | 2 | 8 | 12 | 20 |

---
⁕ if (st >= end) return 0; 부분을 먼저 체크하자. 인덱스가 같거나 역전되면 더 이상 쪼개지 않고 0을 돌려준다는 뜻이다.
⁕ 자식들의 결과(max 값)가 다 나온 뒤에 자기 자신(a[mid])과 합쳐서 위로 전달하는 방식이다.
⁕ 정수 나눗셈이므로 (0+1)/2는 0.5가 아니라 0이 된다.
