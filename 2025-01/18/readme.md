## insert 함수로 리스트 생성 (i=1~5)
- insert 함수는 새로운 노드를 항상 맨 앞(Head)에 붙인다. (전위 삽입)
- 1 삽입: 1 -> NULL
- 2 삽입: 2 -> 1 -> NULL
- 3 삽입: 3 -> 2 -> 1 -> NULL
- 4 삽입: 4 -> 3 -> 2 -> 1 -> NULL
- 5 삽입: 5 -> 4 -> 3 -> 2 -> 1 -> NULL (최종 리스트 상태)


## reconnect(head, 3) 호출 (노드 이동)
- 리스트에서 값이 3인 노드를 찾아 맨 앞으로 옮긴다.

1. 탐색: while 루프를 통해 curr가 3인 노드를 찾는다.
    - prev는 4를 가리키고, curr는 3을 가리키는 상태에서 멈춘다.

2. 연결 끊기: prev->next = curr->next;
    - 4의 다음 노드를 3의 다음 노드인 2로 연결한다. (4 -> 2)

3. 맨 앞으로 이동: curr->next = head;
    - 3의 다음 노드를 기존의 머리였던 5로 연결한다. (3 -> 5)

4. 머리(Head) 갱신: head = curr;
    - 이제 리스트의 시작점은 3이 된다.


## 최종 리스트 상태 및 출력
- 재배치가 완료된 리스트의 연결 순서는 다음과 같다.
- 3 -> 5 -> 4 -> 2 -> 1 -> NULL
- main 함수의 마지막 printf 루프에서 이 값들을 순서대로 출력하므로 결과는 35421이 된다.

---
⁕ new->next = head; head = new; 구조를 보면 데이터가 들어온 순서의 역순으로 리스트가 구성된다는 것을 바로 파악해야 한다.
⁕ 특정 노드를 중간에서 뺄 때는 prev 노드의 역할이 매우 중요하다.
⁕ prev->next를 curr->next로 넘겨줌으로써 중간 노드(3)를 리스트 흐름에서 고립시키는 과정이 핵심이다.
⁕ reconnect 함수의 첫 줄(head->value == value)처럼 찾으려는 값이 이미 맨 앞에 있다면 아무 일도 하지 않고 반환한다.
