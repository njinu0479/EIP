## 문자열 배열과 숨겨진 \0
```c
char str[] = "REPUBLICOFKOREA";
```
C언어에서 문자열을 큰따옴표로 선언하면, 
<br/>눈에는 보이지 않지만 컴퓨터는 문자열이 끝났다는 것을 알기 위해 맨 끝에 항상 \0 를 몰래 하나 추가한다.


## 문자열의 길이(끝) 찾기
```c
int a = 0;
while (str[a] != '\0')
    ++a;
```
현재 위치의 글자(str[a])가 널 문자(\0)가 아닐 때까지 번호(a)를 1씩 증가시킨다.
<br/>a가 15가 되는 순간 str[15]는 \0이므로 반복문이 멈춘다.
<br/>루프가 끝난 직후 a의 최종 값은 15이다.


## putchar 출력
```c
putchar(str[a - 2]);
```
* putchar는 단일 문자 1개를 출력해 주는 함수이다.
* 앞서 구한 a의 값 15를 괄호 안에 대입한다.
* str[15 - 2] -> str[13]
* 위의 표에서 13번 인덱스에 있는 글자를 찾아보면 E이다.

## 최종출력결과
E
<br/>
<br/>⁕ C언어 문자열 끝에는 반드시 \0이 들어가며, 배열 인덱스는 0부터 시작한다.
