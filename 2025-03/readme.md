## 구조체 배열 초기화
```c
struct Test test[] = {{1, "AB"}, {2, "DC"}, {3, "EB"}};
```
test[0] = AB, test[1] = DC, test[2] = EB


## 구조체 포인터 할당
```c
struct Test *p = &test[1];
```
포인터 변수 p에 test[1]의 주소값(&) 넣기
<br/>p는 이제 test[1]을 가리키고 있으므로, p->i는 2이고, p->g는 DC를 가리키게 됨.

```c
printf("%s", p->g + (p->i - 1));
```
1. p->i - 1 계산: p->i는 2이므로, 2 - 1이 되어 결과는 1이다.
2. p->g + 1 계산: p->g는 문자열 DC의 첫 번째 글자인 D의 메모리 주소를 가리키고 있다. 여기에 + 1을 하면 포인터가 한 칸 이동하여 다음 글자인 C의 메모리 주소를 가리키게 된다.
3. %s 출력: %s 서식 지정자는 포인터가 가리키는 위치부터 널 문자(\0)가 나올 때까지 문자열을 출력한다. 현재 포인터가 C를 가리키고 있으므로 C부터 끝까지 출력한다.

## 최종출력결과
C
<br/>
<br/>⁕ %s는 원래 주소값을 받기 때문에 그 주소의 끝 값까지 정상적으로 출력된다.
<br/>⁕ 만약 %c고 역참조를 하지 않으면 주소값을 보기 때문에 이상한 값이 나온다.
