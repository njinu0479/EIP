## 구조체 초기화 및 연결
```c
struct Node t1 = { 0, 5u };
struct Node t2 = { 0, 7u };
struct Node t3 = { 0, 11u };

t3.next = &t2;
t2.next = &t1;
```
Node 구조체는 다음 노드를 가리키는 포인터(next)와 숫자(x)를 가진다.
<br/>코드를 실행하면 노드들이 꼬리에 꼬리를 무는 형태로 연결된다.


## while 반복문
```c
struct Node* curr = &t3;
int sum = 0;

while (curr) {
    sum = sum * 3 + curr->x;
    curr = curr->next;
}
```
curr(현재 노드) 포인터가 t3부터 시작해서 하나씩 다음 노드로 넘어가며 계산을 수행한다.
1. 1회차 / t3 가리킴 / curr -> x = 11 / 0 * 3 + 11 = 11
2. 2회차 / t2 가리킴 / curr -> x = 7 / 11 * 3 + 7 = 40
3. 3회차 / t1 가리킴 / curr -> x = 5 / 40 * 3 + 5 = 125
4. 4회차 / 0 (NULL) / curr -> x = - / 반복문 종료
따라서 반복문이 끝난 후 sum의 값은 125이다.

## 비트 연산자(XOR, ^)와 최종 계산
```c
sum = (sum ^ 42u) + 100u;
```
기호 ^는 제곱이 아니라 XOR 비트 연산자이다.
<br/>두 숫자를 이진수(2진법)로 바꾼 뒤, 각 자릿수가 서로 다를 때만 1을 반환한다.
* sum 값인 125를 이진수로 변환: 0111 1101
* 숫자 42를 이진수로 변환: 0010 1010
-> 이제 각 자리를 비교하며 XOR 연산을 한다.

```
  0111 1101  (125)
^ 0010 1010  (42)
-----------------
  0101 0111  (87)
```
이제 마지막으로 100을 더해준다.
</br>최종 sum = 87 + 100 = 187

## 최종출력결과
187
<br/>
<br/>⁕ 코드에 있는 5u, 42u 같은 곳에서 u는 unsigned(부호 없는 양의 정수)를 의미할 뿐, 
<br/>⁕ 덧셈 곱셈 같은 사칙연산 계산 자체에는 영향을 주지않는다.
