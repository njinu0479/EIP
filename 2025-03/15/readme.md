## 첫 번째 삼항 연산자
```c
z = y % 3 < 3 ? 2 : 1;
```
1. y % 3 계산: $4 \div 3$의 나머지는 1이다.
2. 조건문 확인: 1 < 3은 참(True)이다.
3. 조건이 참이므로 앞의 값인 2가 선택된다.

## 비트 연산 (AND 및 Shift)
```c
z = z & z >> 1;
```
이 부분은 이진수로 변환해서 생각해야 합니다. 현재 z는 2이다.<br/>
1. z >> 1: 2를 오른쪽으로 1비트 이동시킨다.
* 2는 이진수로 0010이다.
* 오른쪽으로 한 칸 밀면 0001 (10진수로 1)이 된다.
2. z & 1: 2와 1을 비트 AND 연산한다.
* 0010 (2) & 0001 (1) = 0000 (결과는 0)
* z = 0

## 두 번째 삼항 연산자 및 논리 연산
```c
z = x > 5 && z <= 3 ? z * x : z / x;
```
1. 조건문 확인
* x > 5: 7 > 5는 참(1)이다.
* z <= 3: 0 <= 3은 참(1)이다.
* 1 && 1: 논리 AND 결과는 참(1)이다.

2. 조건이 참이므로 z * x를 수행
* 0 * 7 = 0
* 최종 z = 0
