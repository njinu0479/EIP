## 구조체 초기화 및 연결
data[0]: [3, 5, 2, 4, 1]
<br/>data[1]: [4, 5, 1]
<br/>data[2]: [4, 4, 1, 5, 4]
<br/>data[3]: [4, 5]

Node 구조체는 다음 노드를 가리키는 포인터(next)와 숫자(x)를 가진다.
<br/>코드를 실행하면 노드들이 꼬리에 꼬리를 무는 형태로 연결된다.


## 루프 분석
```c
for index, lis in enumerate(data):
```
enumerate는 리스트의 인덱스(번호)와 값(내용물)을 동시에 꺼내주는 함수이다.
* index: 0, 1, 2, 3 순서로 번호가 매겨진진다.
* lis: 각 번호에 해당하는 내부 리스트가 하나씩 들어온온다.

## 계산 및 저장 과정 추적
각 순서에서 sum(lis)(합계)와 len(lis)(개수)를 구해 딕셔너리에 저장한다.
1. 1회 / index = 0 / lis = [3, 5, 2, 4, 1] / list_sum = 15
2. 2회 / index = 1 / lis = [4, 5, 1] / list_sum = 10
3. 3회 / index = 2 / lis = [4, 4, 1, 5, 4] / list_sum = 18
4. 4회 / index = 3 / lis = [4, 5] . list_sum = 9

## 최종출력결과
{0: (15, 5), 1: (10, 3), 2: (18, 5), 3: (9, 2)}
<br/>
<br/>⁕ enumerate 함수: 인덱스가 0부터 시작한다.
<br/>⁕ (15, 5)처럼 소괄호로 묶인 것은 수정 불가능한 튜플입니다. 출력 시 소괄호와 콤마를 정확히 적어주는 것이 중요하다.
<br/>⁕ 딕셔너리 괄호: 딕셔너리는 반드시 중괄호 {}를 사용하며, key: value 쌍으로 표현되는 형식이다.
